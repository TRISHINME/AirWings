# ✈️ AirWings - Flight Booking System

A simple airline booking web application built while exploring the fundamentals of the **MERN Stack** and modern web development.

---

# 📖 About the Project

AirWings started as a personal project to understand how different parts of a full-stack application work together. The goal wasn't to build the next airline booking platform—it was to experiment, learn, and gain hands-on experience with concepts that are commonly used in MERN development.

Although the project is fairly simple, it covers several important ideas like creating user-friendly interfaces, connecting different pages, handling user interactions, and organizing a project in a clean and scalable way.

Everything in this repository was built from scratch as part of my own learning process, helping me understand not just *how* things work, but *why* they're designed that way.

# ▶️ Running the Project

1. Clone the repository.

```bash
git clone https://github.com/your-username/airwings-flight-booking-system.git
```

2. Install the required dependencies.

```bash
npm install
```

3. Start the development server.

```bash
npm run dev
```

4. Open the application in your browser and explore the booking flow.
